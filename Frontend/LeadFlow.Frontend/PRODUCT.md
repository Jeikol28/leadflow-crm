# Product

## Register

brand

## Users

LeadFlow esta pensado para pequenas y medianas empresas, equipos comerciales, vendedores, gerentes y administradores que necesitan ordenar clientes, oportunidades, tareas, cotizaciones y reportes sin depender de hojas de calculo dispersas.

## Product Purpose

LeadFlow centraliza el proceso comercial completo: registro de clientes, pipeline de leads, tareas de seguimiento, historial de interacciones, cotizaciones con IVA 13% para Costa Rica, reportes, alertas, seguridad multiempresa y asistencia de IA. El objetivo es ayudar a vender mejor, priorizar oportunidades y cerrar ventas con menos trabajo manual.

## Brand Personality

Profesional, inteligente y confiable. La experiencia debe sentirse premium y moderna, pero no exagerada: clara para empresas reales, pulida para venderse como SaaS, y suficientemente distintiva para no parecer una plantilla generica hecha por IA.

## Anti-references

Evitar landing pages genericas con gradientes morados, tarjetas repetidas sin jerarquia, textos vagos, dashboards falsos que no coinciden con el producto, botones con bajo contraste, animaciones decorativas sin proposito y estilos que parezcan una plantilla comun de IA.

## Design Principles

1. Mostrar producto real antes que promesas abstractas.
2. Usar el pipeline, la IA, las cotizaciones, las alertas y los reportes como prueba visual del valor.
3. Mantener una sensacion premium con jerarquia clara, buen contraste y movimiento sutil.
4. Disenar para confianza empresarial: seguridad, multiempresa, auditoria y datos claros.
5. Hacer que cada seccion explique una accion concreta del sistema.

## Accessibility & Inclusion

La landing debe mantener contraste legible, estados interactivos claros, navegacion responsive y soporte para usuarios con movimiento reducido. Las animaciones deben mejorar la lectura, no ocultar contenido ni bloquear el uso.
